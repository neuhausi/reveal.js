Visualizing Networks with CanvasXpress

Isaac Neuhaus
More examples at CanvasXpress

Features

Open source JavaScript library to visualize Genetic Networks

Used in a web environment or as an R package

Support for many standard data formats

Rich interface to dynamically overlay experimental data

Built for Reproducible Research in mind

CanvasXpress within R-Studio

Used at the console as well as in a Shiny context

Supported Data Formats

  • Native - JSON object
  • GML - Graph Modelling Language
  • XML
    • GPML - Wikipathways
    • KGML - Kegg
    • XGML
      • Cytoscape
      • Metabase

Native JSON Format

            
var cX = new CanvasXpress("canvasId", {
      nodes: [
        {
          id: "Node1",
          color: "rgb(255,0,0)",
          shape: "circle",
          size: 1
        }, {
          id: "Node2",
          color: "rgb(0,255,0)",
          shape: "square",
          size: 1.5
        }, {
          id: "Node3",
          color: "rgb(0,0,255)",
          shape: "triangle",
          size: 2
        }
      ],
      edges: [
        {
          id1: "Node1",
          id2: "Node2",
          color: "rgb(125,125,0)"
        }, {
          id1: "Node1",
          id2: "Node3",
          color: "rgb(0,125,125)"
        }, {
          id1: "Node2",
          id2: "Node3",
          color: "rgb(125,0,125)"
        }
      ]
    }, {
      graphType: "Network"
    }
);

Parameters:

(1) DOM id, (2) data, (3) config

XML is loaded from a URL

            
var cX = new CanvasXpress("canvasId",
    "https://path/to/file/hsa05222.xml", {
      graphType: "Network"
    }
);
